# 🚗 Car Sales Dashboard (Power BI Project)

## 📌 Overview
This project focuses on analyzing car sales data using Power BI to generate meaningful insights and support business decision-making.

The dashboard provides interactive visualizations and tracks key performance indicators (KPIs) such as sales trends, average price, and cars sold.

---

## 🎯 Objective
To design and develop a dynamic Power BI dashboard that helps track sales performance, identify trends, and improve data-driven decisions.

---

## 📊 Key KPIs
- Year-to-Date (YTD) Total Sales  
- Month-to-Date (MTD) Total Sales  
- Year-over-Year (YOY) Growth  
- YTD vs Previous Year (PTYD) Comparison  
- Average Price Analysis  
- Cars Sold Metrics  

---

## 📈 Dashboard Features
- Weekly Sales Trend (Line Chart)  
- Sales by Body Style (Pie Chart)  
- Sales by Color (Pie Chart)  
- Regional Sales Distribution (Map)  
- Company-wise Sales Table  
- Detailed Sales Data Grid  

---

## 🛠 Tools Used
- Power BI  
- MS Excel  
- Data Visualization Techniques  

---

## 🔍 Key Insights
- SUV category contributed the highest sales volume.  
- Certain regions showed significantly higher sales performance.  
- Seasonal trends observed in weekly sales data.
